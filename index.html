API SERVICES
NOMINATIM
Address autocomplete — powered by OpenStreetMap
✓ FREE
Nominatim is OpenStreetMap's free geocoding service — no key, no account, no cost ever. It converts addresses you type into coordinates so the route calculator can work. Used by Apple Maps, Facebook and hundreds of other apps. Limited to 1 request per second which is plenty for personal use.
GOOGLE MAPS
Route distance + driving calculations
NOT SET
What it does: Calculates driving distance and route between two locations.

APIs to enable in Google Cloud Console:
1. Go to console.cloud.google.com
2. Select your project → APIs & Services → Library
3. Search and enable Routes API ← for route distance
4. Search and enable Geocoding API ← for address → coordinates
5. Go to Credentials → Create API Key
6. Copy and paste below

⚠ NOT needed (don't enable these):
— Places API (needs billing, replaced by Nominatim)
— Distance Matrix API (legacy, Routes API does this better)
— Maps JavaScript API (not used)

Requires billing enabled on your Google Cloud project. Free trial gives $300 credit. Personal use stays well within free monthly limits.

APPLE SHORTCUTS — SHARE FROM MAPS
OPTION 1 — CLIPBOARD (EASIEST)
In Google Maps → plan route → tap Share → Copy Link
Open Fuel Log → Range Calculator → tap 📋 PASTE FROM GOOGLE MAPS

OPTION 2 — APPLE SHORTCUT (ONE-TIME SETUP)
1. Open the Shortcuts app on your iPhone
2. Tap + to create new shortcut
3. Add action: "Receive" → Any
4. Add action: "Get URLs from Input"
5. Add action: "Open URL" → set URL to:
fuellog://range?url=[URLs]
6. Tap Add to Share Sheet

Then in Google Maps: Share → Fuel Log shortcut → app opens with route pre-filled.

Note: URL scheme requires the app to be served (not just opened as a file). The clipboard method works for local files.
